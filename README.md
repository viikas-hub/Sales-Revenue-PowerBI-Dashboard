# Sales-Revenue-PowerBI-Dashboard
# 📊 Sales & Revenue Analysis Dashboard (Power BI)

## 🎯 Project Objective
The goal of this project is to analyze raw sales data, clean it using Power Query, create DAX measures, and build an interactive dashboard to track Key Performance Indicators (KPIs) like Revenue, Cost, and Profit.

## 🛠️ Tech Stack & Tools Used
- **Tool:** Power BI Desktop
- **Data Cleaning:** Power Query Editor (Data type correction, handling null values)
- **Calculations:** DAX (Data Analysis Expressions) for custom KPI measures
- **Visualizations:** KPI Cards, Line Chart (Time-series trend), Clustered Bar Chart, and Slicers (Interactive Filters)

## 💡 Key Business Insights
- **Top Channels:** Online sales outperform retail and partner channels significantly.
- **Top Category:** Electronics is the leading revenue-generating product category.
- **Performance:** Interactive filters allow managers to check region-specific performance (e.g., Europe, North America) instantly.
